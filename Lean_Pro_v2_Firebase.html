LEAN MANAGEMENT — CORE RULES
LM-1
Daily standup: 8:00–8:15 AM sharp. All supervisors mandatory.
LM-2
Production board updated daily by 9:00 AM. Data falsification = disciplinary offense.
LM-3
Every team of 5 workers: minimum 1 Kaizen idea per month.
LM-4
KPI below target 3 consecutive days → mandatory PDCA session same day.
LM-5
5S: 10 min before shift + 10 min after. No production before 5S.
LM-6
WIP must never exceed agreed Kanban limits.
LM-7
Every operator trained in minimum 2 operations. Skills matrix updated monthly.
LM-8
Monthly Lean audit: minimum score 80/100.
INDUSTRIAL ENGINEERING — PROCESS RULES
IE-1
Time study: minimum 10 cycles. Trained operator. IE officer present.
IE-2
SAM updated within 48 hours of time study.
IE-3
Line balance efficiency must exceed 80%.
IE-4
Bottleneck identified before style change.
IE-5
Efficiency target: ≥75%. New style ramp-up: 60%→70%→75%.
IE-6
VSM created for every new style at planning stage.
IE-7
Skill rating updated every 3 months. Signed by operator and supervisor.
IE-8
No style enters production without line plan, allocation, and SAM confirmed.
QUALITY CONTROL — ALL RULES
QC-1
QC checks every 10th piece. Tally updated hourly.
QC-2
Defective pieces returned to operator immediately. No defect passes forward.
QC-3
3 same-type defects in 30 min = LINE STOP.
QC-4
AQL 2.5 final inspection before every shipment.
QC-5
5-Why RCA for any defect more than 3 times in one shift.
QC-6
Critical defects → STOP ALL PRODUCTION. Management informed in 15 min.
QC-7
4-point inspection on every fabric roll received.
THE 20 GOLDEN RULES — EVERY EMPLOYEE, EVERY DAY
RULE 01
Never pass a defect to the next station. Stop. Fix. Report.
RULE 02
Never overproduce. Make only what is needed, when it is needed.
RULE 03
Always follow the SOP. Shortcuts cause defects.
RULE 04
Keep workstation clean always. 5S is not seasonal.
RULE 05
Speak up when you see a problem. Silence is waste.
RULE 06
Never bypass Kanban. It protects the line from chaos.
RULE 07
Always label materials. Clearly and correctly.
RULE 08
Every downtime minute is a problem to solve, not accept.
RULE 09
Report machine problems immediately. Do not hide breakdowns.
RULE 10
Board data must be accurate. No guessing. No manipulation.
RULE 11
Attend daily standup. No exceptions without prior approval.
RULE 12
Submit 1 Kaizen idea per month. Improvement is everyone's job.
RULE 13
Root cause first, then corrective action.
RULE 14
Inventory is waste. Reduce WIP every week.
RULE 15
Cross-training mandatory. Minimum 2 operations per operator.
RULE 16
Buyer standards are minimum. Aim to exceed them.
RULE 17
Submit Waste Observation Cards. Noticing is not enough.
RULE 18
Audit findings corrected within agreed deadline.
RULE 19
No style change without revised SOP and operator briefing.
RULE 20
Respect your team. Lean = Trust + Transparency.
