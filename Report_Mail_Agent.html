6 📊 Google Sheet Backup

Har fetch ka overview (system, status, records) ek Google Sheet mein history ke roop mein save ho jayega — data safe rahega aur aage trend analysis ke kaam aayega. Setup ek baar karna hai (5 min ka kaam), niche instructions hain.
1. sheets.google.com par jaake ek nayi Sheet banao 2. Extensions → Apps Script kholo 3. Purana code hatakar ye paste karo: function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); data.rows.forEach(function(r){ sheet.appendRow([data.timestamp, r.system, r.status, r.count]); }); return ContentService.createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); } 4. Upar-right "Deploy" → "New deployment" → gear icon se "Web app" select karo 5. "Execute as": Me, "Who has access": Anyone — phir Deploy dabao 6.
